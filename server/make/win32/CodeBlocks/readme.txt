Howto compile the daimonin server using Code::Blocks
1.) Install the Code::Blocks with integrated MinGW.
2.) Dbclk on Daimonin.workspace.
3.) Select from Menue: compile->compile All Projects.
4.) If you get an error like (...)loader.o : No such file (..) just press copile again.
5.) execute the server\install\win32\install.bat
