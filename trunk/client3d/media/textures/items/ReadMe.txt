//===================================================
// Items
//===================================================

The filenames of this items must be written next to the matching gfx-name in client3d/srv_files/bmaps_convert!
Example:
If you want to make a new scroll item gfx, you search in the bmaps_convert file for the "scroll" entry
and put the name of your new scroll next to it:

scroll   item_scroll.png

* All items must be 64x64 pixel ARGB png pictures.
* The background must be transparent.
* Only alpha values of 0% or 100% are alled in the png.
