//===================================================
// Client3d as Visual c++ project HOWTO
//===================================================

A) Install Visual c++ 2005 Express Edition
B) Install Windows Platform SDK
C) Copy "bin", "include" and "lib" from PlatformSDK into \Microsoft Visual Studio\VC\
----
1) Start the Setup.bat
2) Dblclk on the project file
3) Compile (If you are asked to save the project-file -> press save)
4) run
