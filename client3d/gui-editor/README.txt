This is the DAIMONIN gui-editor for the new 3D client. See http://www.daimonin.net for details about the game.

This software is distributed under the terms of the GPL 2, see file LICENSE for details.

Daimonin developers may access the gui-editor's SVN folder at daimonin/client3d/gui-editor.



Configuration
==============

* you need JAVA 1.6 (or 6.0 as Sun calls it) to run the editor.


How to run
==============

Windows: editor3d_start.bat
Linux  : ./editor3d_start
